Cadastro do Dados Gerais

  1. Usuário deverá cadastrar os Dados Gerais do Projeto. São eles:
    • Título do Projeto;
    • Bolsas solicitadas: quantidade de bolsas solicitadas;
    • Ano de referência: ano de referência do projeto;
    • Data limite da seleção de monitores: clicar na figura ao lado para selecionar a data limite para seleção de monitores;
    • Departamento: selecionar dentre os departamentos cadastrados;
    • Edital: selecionar dentre os editais cadastrados;
    • Email oficial do projeto;
  2. Usuário deverá preencher as informações: Descrição, Diagnóstico e Objetivos, Estratégias, Metodologia de Avaliação. Para tanto, o usuário deverá clicar sobre os respectivos termos, clicar no espaço reservado a inserção de texto e digitar as informações correspondentes ao termo digitado no espaço destinado ao seu texto (Dica: ficará em amarelo o espaço reservado a digitação quando o usuário estiver inicialmente clicado no referido espaço para digitiação);
  3. Usuário deverá clicar em um dos botões:

    • Cancelar: cancela a operação realizada (cadastro de projeto de monitoria);

    • Gravar Proposta: grava os dados fornecidos para que o usuário possa continuar o cadastramento da proposta a partir dos dados já fornecidos;

    • Componentes Curriculares >>: vai para o próximo passo no preenchimento das informações do projeto, ou seja, a adição dos componentes curriculares;
