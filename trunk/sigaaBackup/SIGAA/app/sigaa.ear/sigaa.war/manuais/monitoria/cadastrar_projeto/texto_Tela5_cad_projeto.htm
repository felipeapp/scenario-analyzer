Seleção de Coordenador do Projeto

  1. Sistema exibirá os docentes orientadores do projeto de Monitoria;
  2. Usuário deverá selecionar qual dos docentes será coordenador do projeto;
  3. Usuário deverá clicar em um dos seguintes botões:
    • << Docentes Envolvidos: retorna para a tela anterior;
    • Cancelar: cancela a operação realizada (cadastro de projeto de Monitoria);
    • Resumo >>: permite continuar o preenchimento das informações do projeto;
