Cadastro de turma - Passo 4

  1. Usuário deverá ler os dados cadastrados para detectar possíveis falhas;
  2. Clicar em um dos seguintes botões:
    • Confirmar: confirma e finaliza o cadastro de turma;
    • Cancelar: cancela a operação realizada (cadastro de turma);
    • << Horários: retorna para a tela de Horários;
    • << Alterar Docentes: retorna para a tela de Docentes;
    • << Dados Gerais: retorna para a tela de Dados Gerais;
