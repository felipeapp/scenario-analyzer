Alteração de discente

  1. A partir do resultado da busca (parte inferior da tela anterior), o usuário deverá clicar na figura (Alterar Discente na legenda da tela anterior) correspondente ao aluno que deseja alterar;
  2. Sistema exibirá os dados do referido aluno;
  3. Usuário alterará os dados que desejar. (Para maiores informações sobre os dados do discente, consultar manual de cadastro de discente);
  4. Após a alteração dos dados que o usuário desejar, ele deverá clicar no botão Confirmar. Este botão estará presente na tela que exibir os dados do discente;
