Passos para busca de proposta de curso lato sensu

  1. Usuário deverá buscar a proposta utilizando um dos critérios:
    • Curso: nome do curso ao qual se deseja buscar a proposta;
    • Todos: todas as propostas cadastradas;
  2. Usuário deverá clicar no botão Buscar;
  3. Sistema exibirá lista de propostas em conformidade com o critério de busca, tal como mostrado na tela anterior;
