Passos para inclusão dos dados acadêmicos do discente

  1. Usuário deverá cadastrar os Dados Acadêmicos do discente. São eles:
    • Curso: selecionar dentre os cadastrados;
    • Turma de entrada: selecionar dentre as disponíveis. Indica a qual turma de entrada aquele aluno fará parte;
    • Forma de ingresso: selecionar dentre as opções disponíveis;
    • Procedência do aluno: selecionar dentre as opções disponíveis (própria IES, outra IES, etc). Indica de onde o aluno veio;
    • Ano-Período de ingresso: há dois espaços para preenchimento dessa informação. No primeiro deve ser preenchido o ano de ingresso e no segundo o período de ingresso, por exemplo 2007 - 01;
    • Observação: espaço destinado a digitação de qualquer observação referente ao aluno;
  2. Usuário deverá clicar em um dos botões:
    • << Dados pessoais: retorna para a tela de Dados Pessoais;
    • Cancelar: cancela a operação realizada (cadastro de discente);
    • Próximo >>: permite continuar o preenchimento das informações do discente;
