Cadastro de Membros do Projeto

  1. Nesta tela, o usuário informará quem são os participantes do projeto. Para cada categoria de participante - Docente, Discente, Técnico ou Externo - as informações são diferentes, conforme veremos a seguir:
    • Docente
      • Informar se o docente é da UFRN ou apenas da unidade do usuário;
      • Nome do docente;
      • Informar o tipo de participação do docente no projeto (coordenador, vice-coordenador ou colaborador) ;
      • Informar a carga horária (CH) que o docente dedicará ao projeto em horas;
    • Discente
      • Informar se o discente é de Graduação, Mestrado ou Doutorado;
      • Nome do discente;
      • Informar a carga horária (CH) que o discente dedicará ao projeto em horas;
    • Servidor Técnico-Administrativo:
      • Nome do servidor técnico-administrativo;
      • Informar a carga horária (CH) que este servidor dedicará ao projeto em horas;
    • Externo:
      • Nome do membro externo;
      • CPF;
      • Sexo;
      • Formação: selecionar entre as cadastradas;
      • Instituição de origem: selecionar entre as cadastradas;
      • Informar a carga horária (CH) que o membro externo dedicará ao projeto em horas;
  2. Após a informação dos dados de quaisquer das categorias, o usuário deverá clicar no botão Adicionar Membro;

    3. Vale lembrar que após a inserção de um membro ao projeto, ele aparecerá na Lista de Membros;

    Caso o usuário deseje excluir algum membro dessa lista, deverá clicar no X respectivo a tal membro;

    Para adicionar outros docentes, o usuário deverá refazer o passos 1 e 2, ou seja, infomar os dados do membro, conforme sua categoria, e clicar no botão Adicionar Membro;

  3. Usuário deverá clicar em um dos seguintes botões:
    • << Informar Financiamentos Externos: retorna para a tela anterior;
    • Gravar: permite que o usuário continue o cadastramento em momento posterior a partir dos dados cadastrados até aqui;
    • Cancelar: cancela a operação realizada (cadastro de projeto de pesquisa externo);
    • Definir Cronograma>>: permite continuar o preenchimento das informações do projeto;
