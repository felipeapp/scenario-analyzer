Dados para cadastro de Projeto de Pesquisa Externo

  1. Os dados necessários ao cadastramento de um Projeto de Pesquisa Externo estão descritos a seguir. Vale lembrar que os dados obrigatórios estão em vermelho. São eles:
    • Dados iniciais
      • Título
      • Centro
      • Período do Projeto
        • Data de início
        • Data de fim
      • Edital
      • Palavras-chave
      • Email
      • Área de Conhecimento
        • Grande área
        • Área
        • Sub-área
        • Especialidade;
      • Grupo de Pesquisa (caso projeto ligado a um grupo);
      • Linha de Pesquisa
      • Arquivo do projeto
    • Descrição detalhada do projeto
      • Descrição resumida do projeto
      • Introdução
      • Objetivos
      • Justificativa
      • Metodologia
      • Bibliografia
    • Financiamentos
      • Entidade Financiadora
      • Natureza do financiamento (auxílio financeiro, auxílio financeiro + bolsa, bolsa, outro). Se a natureza do financiamento incluir bolsa, informar: modalidade, tipo e quantidade de bolsas;
      • Data de início
      • Data de fim
    • Membros do Projeto
      • Docente
        • Nome
        • Tipo de participação (coordenador, vice, colaborador)
        • Carga horária
      • Discente
        • Nome
        • Carga horária
      • Técnico Administrativo
        • Nome
        • Carga horária
      • Externo
        • Nome
        • CPF
        • Sexo
        • Formação (especialização, mestre, doutor,...)
        • Instituição de origem
        • Carga horária
    • Cronograma
      • Nome da atividade (ao menos 1 atividade deve ser cadastrada);
      • Meses nos quais a atividade será realizada;
