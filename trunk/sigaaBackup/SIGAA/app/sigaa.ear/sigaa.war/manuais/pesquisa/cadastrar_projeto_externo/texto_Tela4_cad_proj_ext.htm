Financiamentos

  1. Nesta tela, serão cadastrados os financiamentos do projeto. Para cada financiamento o usuário deverá informar:
    • Entidade Financiadora (selecionar entre as opções cadastradas);
    • Natureza do Financiamento (Auxílio financeiro, Auxílio financeiro + bolsa, Bolsa, outro). Caso a natureza do financiamento inclua bolsa, deverão ser informados também os dados da bolsa (modalidade, tipo e quantidade de bolsas). Após incluir essas informações, o usuário deverá clicar no botão Adicionar Bolsa Externa;
    • Data de início: clicar na figura respectiva ao calendário para selecionar data;
    • Data de fim: clicar na figura respectiva ao calendário para selecionar data;
  2. Após a informação dos dados do financiamento, o usuário deverá clicar no botão Adicionar Financiamento;

    3. Vale lembrar que após a inserção de um financiamento, ele aparecerá na Lista de Financiamentos;

    Caso o usuário deseje excluir algum financiamento dessa lista, deverá clicar no X respectivo a tal financiamento;

    Para adicionar outros financiamentos, o usuário deverá refazer o passos 1 e 2, ou seja, infomar os dados do financiamento e clicar no botão Adicionar Financiamento;

  3. Usuário deverá clicar em um dos seguintes botões:
    • << Alterar Detalhes do Projeto: retorna para a tela anterior;
    • Gravar: permite que o usuário continue o cadastramento em momento posterior a partir dos dados cadastrados até aqui;
    • Cancelar: cancela a operação realizada (cadastro de projeto externo);
    • Definir Colaboradores>>: permite continuar o preenchimento das informações do projeto;
