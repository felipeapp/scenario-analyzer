Cadastro de Docentes

  1. Usuário deverá fornecer informar o nome do docente;
  2. Selecionar o(s) componente(s) curricular(es) relacionados ao docente. Para selecionar mais de um componente, deve-se manter a tecla CTRL pressionada enquanto os componentes são selecionados;
  3. Clicar no botão Adicionar Docente ao Projeto;

    4. Vale lembra que após incluir um docente ao projeto de Monitoria, o nome dele, bem como os componentes nos quais ele está envolvido aparecerão na Lista de Orientadores e seus Componentes Curriculares.

    Caso o usuário deseje excluir algum docente dessa lista, deverá clicar no X respectivo ao docente que deseja excluir;

    Para adicionar outros docentes, o usuário deverá refazer o passos 1, 2 e 3, ou seja, infomar o nome do docente, componentes nos quais está envolvido e clicar no botão Adicionar Docente ao Projeto;

  4. Usuário deverá clicar em um dos seguintes botões:
    • << Componentes Curriculares: retorna para a tela anterior;
    • Cancelar: cancela a operação realizada (cadastro de projeto de Monitoria);
    • Gravar Proposta: permite que o usuário continue o cadastramento em momento posterior a partir dos dados cadastrados até aqui;
    • Seleção de Coordenador>>: permite continuar o preenchimento das informações do projeto;
