Cadastro de Componentes Curriculares

  1. Usuário deverá fornecer Código ou Nome do componente curricular que deseja adicionar ao projeto de monitoria e clicar no botão Adicionar Componente Curricular;

    2. Vale lembrar que após ter sido adicionado, o componente curricular aparecerá na Lista de componentes curriculares do projeto. Deverão ser informados: o(s) semestre(s) de oferecimento do componente; se ele é um componente que deve ter sido obrigatoriamente cursado pelo monitor e este obtido média maior do que sete e, por fim, o plano de trabalho do aluno relacionado ao componente curricular em questão.

    Caso o usuário deseje remover um componente curricular, deverá clicar no X respectivo ao componente que deseja excluir;

    Para adicionar outros componentes curriculares ao projeto, retornar ao passo 1, ou seja, informar novamente Código ou Nome do componente e clicar no botão Adicionar Componente Curricular, informar o(s) semestre(s) de oferecimento do componente, se ele é obrigatório na seleção de monitores e seu plano de trabalho;

  2. Usuário deverá clicar em um dos botões:
    • << Dados do Projeto: retorna para a tela dos dados do projeto;
    • Cancelar: cancela a operação realizada (cadastro de projeto de monitoria);
    • Gravar Proposta: Grava os dados fornecidos para posterior finalização do cadastro da proposta, sem que haja necessidade de recadastramento dos dados já cadastrados pelo usuário;
    • Docentes Envolvidos >>: vai para a tela de cadastro dos docentes envolvidos no projeto;
