Confirmação de Cadastro

  1. Usuário deverá ler os dados cadastrados para detectar possíveis falhas;
  2. Usuário deverá clicar em um dos seguintes botões:
    • Cancelar: cancela a operação realizada (cadastro de projeto de monitoria);
    • <<Docentes Envolvidos: retorna para a tela anterior;
    • Gravar Proposta: grava as informações fornecidas, possibilitando edição posterior;
    • Finalizar Edição e Enviar para Departamentos: envia as informações fornecidas aos Departamentos de forma definitiva, ou seja, não permite mais edição;
