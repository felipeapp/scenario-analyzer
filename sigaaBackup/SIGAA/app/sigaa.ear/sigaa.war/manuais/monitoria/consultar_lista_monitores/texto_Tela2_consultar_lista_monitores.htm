Busca por monitores

  1. Para realizar a busca, o usuário deverá preencher um dos filtros:
    • Projeto;
    • Orientador;
    • Situação do monitor;
    • Tipo de vínculo do monitor;
  2. Usuário deverá clicar no botão Buscar;
  3. Sistema exibirá a lista de monitores em conformidade com o filtro escolhido;
  4. Usuário deverá clicar na figura Visualizar Dados do Monitor respectiva ao monitor que se deseja consultar dados;
