Busca por projeto

  1. Usuário deverá realizar a busca do projeto que deseja consultar situação. Para tanto, deverá preencher um dos campos:
    • Título do Projeto;
    • Servidor envolvido;
    • Edital do projeto;
    • Ano do projeto;
    • Situação do projeto;
    • Componente curricular (código ou nome do componente);
  2. Usuário deverá clicar no botão Buscar;
