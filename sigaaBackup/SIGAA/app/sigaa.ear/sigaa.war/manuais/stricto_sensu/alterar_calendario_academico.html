Alterar calendário acadêmico do programa

Após acessar o SIGAA, o usuário deverá entrar no módulo Portal Coord. Stricto. Para realizar a alteração do calendário acadêmico para um determinado período de aulas e para o programa do qual faz parte, os passos abaixo devem ser seguidos:

Na tela 1
  1. Dentre as seis abas dessa tela: Cadastro, Aluno, Turmas, Matrículas, Consultas e Relatórios, posicionar o cursor sobre Cadastro;
  2. Clicar sobre a opção Calendário do programa.
Na tela 2 - Alteração do calendário
  1. No campo Datas para (sigla do programa), clicar sobre uma das seguintes opções:
    • Novo: Para cadastrar um calendário para um novo período.
      1. Preencher os seguintes campos:
        • Ano-Período;
        • Se for o período vigente, clicar sobre o quadrado ao lado do campo Tornar esse calendário vigente;
        • Período Letivo;
        • Consolidação de Turmas;
        • Trancamento de Turmas;
        • Matrícula OnLine;
        • Análise dos Coordenadores/Orientadores da Matrícula;
        • Re-Matrícula;
        • Análise dos Coordenadores/Orientadores para Re-Matrícula.
      2. Clicar sobre o botão Cadastrar Calendário.
    • Período já cadastrado: Para alterar um período já existente no cadastro.
      1. Modificar qualquer um dos campos citados para cadastro de um novo período, exceto o campo Ano-Período;
      2. Clicar sobre o botão Alterar Calendário
