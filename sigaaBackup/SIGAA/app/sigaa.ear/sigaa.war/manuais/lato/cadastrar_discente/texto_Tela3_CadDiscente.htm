Passo para confirmação dos dados do discente

  1. Usuário deverá ler os dados cadastrados para detectar possíveis falhas;
  2. Clicar em um dos seguintes botões:
    • Confirmar: confirma e finaliza o cadastro de discente;
    • Cancelar: cancela a operação realizada (cadastro de discente);
    • << Dados Pessoais: retorna para a tela de Dados Pessoais;
    • << Dados Acadêmicos: retorna para a tela de Dados Acadêmicos;
