Cadastro de Turma - Passo 2

  1. O usuário deverá cadastrar os dados referentes ao docente da turma em questão. São eles:
    • Critério de busca do docente: selecionar o critério de busca do docente (Todos da UFRN, somente da unidade do usário ou externo) através de clique no local respectivo;
    • Nome do docente: informar o nome do docente. Vale lembrar que o sistema exibirá a lista de docentes cadastrados com as iniciais fornecidas pelo usuário, conforme o critério de busca, para que o usuário o selecione, caso deseje;
    • Carga horária do docente dedicada a turma;
  2. Usuário deverá clicar no botão Adicionar Docente;

    3. Caso o usuário deseje adicinar outros docentes a turma, deverá fazer novamente o passo 1 e, em seguida, clicar no botão Adicionar Docente, para cada docente adicionado a turma;

    Vale lembrar que à medida que os docentes forem sendo adicionados a turma, seus nomes vão sendo exibidos abaixo do título DOCENTES ADICIONADOS. Para remover docente adicionado a turma, o usuário deverá clicar no X respectivo ao docente que se deseja excluir;

  3. Usuário deverá clicar em um dos botões:

    • << Dados Básicos da Turma: retorna para a tela de Dados Básicos da Turma;

    • Cancelar: cancela a operação realizada (cadastro de turma);

    • Definir Horários >>: permite continuar o preenchimento das informações da turma (Definir Horários);
