Passos para registro de afastamento

  1. Sistema exibirá tela para que o usuário preencha os dados do afastamento, conforme tela anterior;
  2. Usuário deverá fornecer os seguintes dados:
    • Nome do aluno
    • Tipo do afastamento: escolher uma das opções (Ex: abandono de curso, trancamento de programa, etc);
    • Data do afastamento: data do afastamento (Clicar na figura ao lado da data para escolher uma data);
  3. Usuário deverá clicar em um dos botões:
    • Confirmar: confirma a operação realizada;
    • Cancelar: cancela a operação realizada (registro de afastamento).
