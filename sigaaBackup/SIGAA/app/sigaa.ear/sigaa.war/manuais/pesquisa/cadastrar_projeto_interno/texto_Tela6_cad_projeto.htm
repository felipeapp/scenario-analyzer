Confirmação de Cadastro

  1. Usuário deverá ler os dados cadastrados para detectar possíveis falhas;
  2. Usuário deverá clicar em um dos seguintes botões:
    • << Alterar Cronograma: retorna para a tela anterior;
    • Gravar : grava as informações fornecidas, possibilitando edição posterior;
    • Confirmar cadastro: confirma e finaliza o cadastro;
    • Cancelar cadastro: cancela a operação realizada (submissão de projeto interno).
