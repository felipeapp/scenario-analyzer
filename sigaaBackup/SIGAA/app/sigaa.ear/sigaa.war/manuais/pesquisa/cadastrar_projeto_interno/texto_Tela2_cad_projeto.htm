Cadastro dos Dados Iniciais do Projeto

  1. Usuário deverá cadastrar os Dados Gerais do Projeto. São eles:
    • Título do Projeto;
    • Unidade: sistema exibirá a unidade do usuário;
    • Centro: informar o centro ao qual o projeto estará atrelado;
    • Edital de Pesquisa: selecionar um dos editais em vigência;
    • Palavras-chave: palavras-chave do projeto;
    • E-mail: email do projeto;
    • Área de conhecimento:
      • Grande Área;
      • Área;
      • Sub-área;
      • Especialidade;
    • Grupo e Linhas de Pesquisa
      • Projeto vinculado a grupo de pesquisa: usuário deverá clicar no espaço respectivo ao Sim, se o projeto a ser submetido estiver vinculado a grupo de pesquisa;
      • Grupo de Pesquisa: caso tenha selecionado Sim ao tópico anterior, o usuário deverá informar a qual grupo seu projeto está vinculado;
      • Linha de Pesquisa: digitar a qual linha de pesquisa seu projeto está vinculado;
    • Arquivo do Projeto
      • Caso o usuário queira, poderá submeter arquivo contendo os dados do projeto. Para tanto, deverá clicar no botão Arquivo... e selecionar, dentre os arquivos do seu computador, o arquivo contendo os dados do projeto;
  2. Usuário deverá clicar em um dos botões:

    • Cancelar: cancela a operação realizada (submissão de proposta de projeto interno);

    • Informar Detalhes do Projeto>>: vai para o próximo passo no preenchimento das informações do projeto;
