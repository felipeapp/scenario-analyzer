Descrição Detalhada do Projeto

  1. Usuário deverá fornecer as seguintes informações: Descrição Resumida, Introdução, Objetivos, Justificativa, Metodologia e Bibliografia. Para tanto, o usuário deverá clicar sobre o tópico (por exemplo, Introdução) e digitar a informação respectiva a ele no espaço destinado para tal. Vale lembrar que antes de iniciar a digitação, o usuário deverá clicar no espaço destinado ao texto;
  2. Usuário deverá clicar em um dos botões:
    • << Alterar Dados Iniciais: retorna para a tela dos dados iniciais do projeto;
    • Gravar: Grava os dados fornecidos para posterior finalização do cadastro da proposta, sem que haja necessidade de recadastramento dos dados já cadastrados pelo usuário;
    • Cancelar: cancela a operação realizada (submissão de proposta de projeto interno);
    • Definir Colaboradores>>: vai para a tela de cadastro dos colaboradores envolvidos no projeto;
