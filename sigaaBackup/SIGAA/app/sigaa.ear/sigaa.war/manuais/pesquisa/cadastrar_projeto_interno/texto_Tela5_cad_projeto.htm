Cronograma de Atividades

  1. Para cada atividade do cronograma, o usuário deverá digitar a atividade no espaço abaixo de Atividade e em seguida clicar nos meses respectivos ao cronograma dessa atividade;

    2. Para adicionar nova atividade, o usuário deverá clicar em Adicionar Atividade;

    Para remover uma atividade, o usuário deverá clicar na figura respectiva a atividade a ser removida;

    Para limpar todo o cronograma de uma atividade, o usuário deverá clicar em Limpar Cronograma;

  2. Usuário deverá clicar em um dos seguintes botões:
    • << Definir Colaboradores: retorna para a tela anterior;
    • Gravar: permite que o usuário continue o cadastramento em momento posterior a partir dos dados cadastrados até aqui;
    • Cancelar: cancela a operação realizada (submissão de proposta de projeto interno);
    • Tela de Resumo e Confirmação>>: permite continuar o preenchimento das informações do projeto;
